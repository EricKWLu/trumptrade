**Step 1 — Overwrite `frontend/vite.config.ts`** with the following exact content:
```typescript
import path from "path"
import tailwindcss from "@tailwindcss/vite"
import react from "@vitejs/plugin-react"
import { defineConfig } from "vite"

export default defineConfig({
  plugins: [react(), tailwindcss()],
  resolve: {
    alias: {
      "@": path.resolve(__dirname, "./src"),
    },
  },
  server: {
    proxy: {
      "/api": "http://localhost:8000",
    },
  },
})
```

**Step 2 — Update `frontend/tsconfig.json`**: Add `"baseUrl"` and `"paths"` inside `compilerOptions`. The file already exists from create-vite; merge these two keys into the existing `compilerOptions` object:
```json
"baseUrl": ".",
"paths": {
  "@/*": ["./src/*"]
}
```

**Step 3 — Update `frontend/tsconfig.app.json`**: Apply the same `baseUrl` and `paths` additions to `compilerOptions` in this file as well.

**Step 4 — Initialize shadcn/ui** (run from the `frontend/` directory):
```bash
cd frontend
npx shadcn@latest init
```
When prompted:
- Style: Default (New York or Default — either is fine)
- Base color: accept default (Neutral or Slate)
- CSS variables: Yes
- If asked about Tailwind version: select v4

shadcn/ui will add `frontend/src/lib/utils.ts` and update `frontend/components.json`. Let it run — do not interrupt.

**Step 5 — Overwrite `frontend/src/main.tsx`** with:
```typescript
import { StrictMode } from "react"
import { createRoot } from "react-dom/client"
import { QueryClient, QueryClientProvider } from "@tanstack/react-query"
import { ReactQueryDevtools } from "@tanstack/react-query-devtools"
import "./index.css"
import App from "./App.tsx"

const queryClient = new QueryClient({
  defaultOptions: {
    queries: {
      staleTime: 1000 * 30,
      retry: 1,
    },
  },
})

createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <QueryClientProvider client={queryClient}>
      <App />
      <ReactQueryDevtools initialIsOpen={false} />
    </QueryClientProvider>
  </StrictMode>,
)
```

**Step 6 — Overwrite `frontend/src/App.tsx`** with a minimal placeholder:
```typescript
export default function App() {
  return (
    <div className="flex min-h-screen items-center justify-center">
      <h1 className="text-2xl font-bold">TrumpTrade</h1>
    </div>
  )
}
```

**Step 7 — Verify the dev server starts** (run from `frontend/` directory, then Ctrl+C after confirming no errors):
```bash
npm run dev
```
