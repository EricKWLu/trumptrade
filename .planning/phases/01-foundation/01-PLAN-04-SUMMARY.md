## Decisions Made
- Used `alembic init --template async` — plain `alembic init` generates a sync env.py incompatible with the `sqlite+aiosqlite://` driver. The async template provides the `run_sync` bridge verbatim.
- Seed data placed in same migration as schema creation — downstream phases rely on these keys at startup; a separate migration would risk a window where the app starts without them.
- `trumptrade.db` is gitignored — already excluded by `.gitignore`; only the migration file (the schema source of truth) is committed.
